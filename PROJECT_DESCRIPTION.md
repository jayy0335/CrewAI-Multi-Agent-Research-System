# CrewAI Multi-Agent Research System

## Overview

A modular AI-powered research and content creation system built with CrewAI framework. This project orchestrates three specialized AI agents that work collaboratively to research topics, analyze trends, and generate comprehensive content.

## What It Does

The system takes a research topic as input and produces:
- **Strategic Research Analysis**: Comprehensive investigation of the topic with key insights and data
- **Trend Analysis Report**: In-depth evaluation of emerging trends, opportunities, and risks
- **Engaging Article**: A well-written, accessible article about the topic's impact on the industry

## The AI Team

1. **Senior Researcher** - Conducts strategic research and coordinates the team
2. **Content Creator** - Analyzes trends and crafts compelling narratives using web search
3. **Writer** - Transforms complex topics into engaging, easy-to-understand articles

## Technology Stack

- **Framework**: CrewAI for multi-agent orchestration
- **LLM**: Groq's Llama 3.3 70B model for high-quality AI responses
- **Tools**: SerperDev for real-time web search capabilities
- **Architecture**: Modular Python package structure for maintainability and scalability

## Key Features

- ✅ **Modular Design**: Separate packages for agents, tasks, configuration, and crew management
- ✅ **Sequential Processing**: Agents work in coordinated sequence for optimal results
- ✅ **Configurable**: Easy to modify agents, tasks, and LLM settings
- ✅ **Extensible**: Simple to add new agents or tasks to expand capabilities
- ✅ **Production-Ready**: Clean code structure following Python best practices

## Use Cases

- Automated research reports on emerging technologies
- Trend analysis for market intelligence
- Content generation for blogs, newsletters, or documentation
- Educational material creation on complex topics
- Competitive analysis and industry insights
